# Flappy-Bird-Nintendo-Switch-port
an unofficial Nintendo Switch port of the popular and nostalgic mobile game, written in Lua using LÖVE Potion

-To play it, you need to use the LÖVE Potion interpreter for Nintendo Switch because is not a standalone nro file, but in the future it will be one

Controls - splash screens

A/plus - skip splash screens

Controls - Menu

A/plus - start game

Controls - In game

A - flap

X - restart game

Y - back to menu

plus/minus - pauses/unpauses

This is an initial release, so let me know the bugs to fix it later!
